# ♻️ E-Waste Management Website

A modern and responsive **E-Waste Management Website** designed to raise awareness about electronic waste, promote responsible recycling, and provide users with information on proper e-waste disposal.

---

## 📌 Project Overview

Electronic waste (E-waste) is one of the fastest-growing waste streams in the world. This website aims to:

- Educate users about e-waste and its environmental impact  
- Promote safe and responsible e-waste recycling  
- Provide information on e-waste collection and management  
- Encourage sustainable and eco-friendly practices  

---

## 🚀 Features

- 🌱 Informative sections about e-waste and pollution  
- 🧠 Awareness content on recycling benefits  
- 📞 Contact section for user inquiries  
- 📱 Fully responsive design (mobile, tablet & desktop)  
- 🎨 Clean and eco-friendly UI design  

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the website  
- **CSS3** – Styling and responsive design  
- **JavaScript** – Interactivity (if applicable)  
- **Font Awesome** – Icons  
- **Google Fonts** – Modern typography
